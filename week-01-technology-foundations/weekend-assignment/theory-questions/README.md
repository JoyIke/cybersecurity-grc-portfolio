## 1. Explain the role of the CPU and why it is called the brain of the computer.

The Central Processing Unit (CPU) performs calculations and executes instructions from programs. Every action a computer performs, such as opening a website or saving a file, involves the CPU processing instructions.

It is called the brain of the computer because it controls how the system operates and directs other components to perform tasks.

---

## 2. Describe the Fetch → Decode → Execute → Store cycle.

The CPU processes instructions through a repeating cycle.

Fetch  
The CPU retrieves the next instruction from memory.

Decode  
The Control Unit interprets the instruction and determines what operation must be performed.

Execute  
The CPU performs the operation.

Store  
The result of the operation is stored in memory for later use.

---

## 3. What is a computer network?

A computer network is a system of connected devices that communicate with each other to share data and resources.

---

## 4. Explain the difference between a router, switch, and hub.

Router  
Directs traffic between different networks using IP addresses.

Switch  
Connects devices within a local network and forwards data to the correct device using MAC addresses.

Hub  
Connects devices but sends data to all devices, which can cause network collisions.

---

## 5. Which protocol assigns IP addresses automatically?

DHCP (Dynamic Host Configuration Protocol) automatically assigns IP addresses to devices when they connect to a network.

---

## 6. Explain the difference between TCP and UDP.

TCP ensures reliable communication by confirming packets arrive in order.

UDP sends data faster but does not guarantee delivery.

---

## 7. What is the TCP/IP model?

The TCP/IP model is a framework that explains how data travels across networks and the internet.

---

## 8. List and explain the four layers of the TCP/IP model.

Application Layer  
Provides services to applications.

Transport Layer  
Manages data transmission between devices.

Internet Layer  
Handles addressing and routing of packets.

Network Interface Layer  
Handles the physical transmission of data across network hardware.

---

## 9. Match these protocols with their functions.

HTTP  
Transfers web pages.

DNS  
Translates domain names into IP addresses.

IP  
Provides addressing and routing.

FTP  
Transfers files between systems.

---

## 10. Which Wi-Fi band offers longer range but lower speed?

2.4 GHz offers longer range but lower speed.
