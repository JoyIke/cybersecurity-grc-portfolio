# Cybersecurity GRC Portfolio

This repository contains coursework, labs, and projects completed during my Risk Management Framework (RMF) cybersecurity training program.

The purpose of this portfolio is to demonstrate foundational cybersecurity knowledge, security analysis skills, and structured documentation aligned with Governance, Risk, and Compliance (GRC) practices.

## Topics Covered

- Technology and networking fundamentals
- Security threats and vulnerabilities
- OWASP attack scenarios
- Risk awareness and security analysis
- Documentation of labs and assignments

Each week contains assignments, supporting artifacts, and version history documentation.
